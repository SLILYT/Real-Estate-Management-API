# Real-Estate-Management-API
Real Estate Management API, emlak ilanlarını yönetmek, aramak, filtrelemek ve emlakçılar ile müşteriler arasındaki iletişimi sağlamak amacıyla geliştirilmiş bir RESTful Web API projesidir.

## Proje Amacı

* Emlak ilanlarını sistematik şekilde yönetmek  
* Gelişmiş filtreleme ve arama özellikleri sunmak  
* Rol bazlı yetkilendirme ile güvenli bir yapı kurmak  
* Gerçek hayata yakın, ölçeklenebilir bir backend mimarisi oluşturmak
